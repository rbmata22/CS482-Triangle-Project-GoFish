## Installations & Setup
1. npm create vite@latest
2. Framework: React
3. Variant: Javascript
4. npm install
5. npm install firebase
6. cd go-fish

## After Vite Project is created and 'go-fish' file exists
1. cd go-fish
2. npm install --legacy-peer-deps
3. npm install firebase --legacy-peer-deps
4. npm run dev
